# ControleEstoque-20240024

API REST para controle de estoque com módulo de vendas desenvolvida em Spring Boot.

## Funcionalidades
- CRUD de Clientes, Produtos, Categorias, Fornecedores
- Registro de vendas com baixa automática no estoque
- Validação de estoque insuficiente com rollback transacional
- Relacionamentos JPA completos

## Tecnologias
- Java 17
- Spring Boot 3.5.6
- Spring Data JPA
- MySQL/MariaDB
- Maven
- Lombok
